The Box Model

HTML elements are rendered by a browser as a set of boxes.

Content Box

The innermost box is known as the content box. That's the part that actually contains content, like text.

Padding

The content box is surrounded by padding. Any background styling applied to an element is applied to both the content box and the padding around it.

Border

The border is a shape (usually rectangular and thin) around an element's padding (or around the content box if the element has no padding).

Margin

An element's margin is the box that separates the element from other elements.

Side–Specific Values

Each of the three box model properties (padding, border, and margin) can be set to have a consistent value on every side of the box, or they can be set one side at a time. For example:

  • margin-top
  • padding-left
  • border-bottom

Shortcuts

While using the side–specific values allows a developer to set each side individually, they are verbose. It is common to use shortcuts to specify values for each side. If multiple values are associated with a single box model property, the first value is for the top, then the right, bottom, and left. If only two values are specified, the first will be applied to the top and bottom, and the second to the right and left.
